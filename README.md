# Gaming-Analytics

This project contains a **gaming analytics CSV dataset** and a **Python-based analysis workflow** using Pandas.

The dataset includes player performance, engagement, game modes, and in-game purchase behavior.  
It is used for **basic exploratory data analysis and statistical analysis** in **Google Colab**.

## Tools Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Google Colab  

> This repository has been **modified according to the structure of the CSV file** used for analysis.
